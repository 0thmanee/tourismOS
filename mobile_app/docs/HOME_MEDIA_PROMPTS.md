# Home Screen Image Prompts

Use these prompts to generate all Home visuals as realistic, premium, mobile-first images.

## Brand and style lock (apply to every prompt)

- Cinematic but natural light, realistic photography, no CGI look.
- Color mood aligned with app palette: deep forest green, warm gold accents, soft neutral highlights.
- Human-centered framing, rich texture, and clear depth.
- Portrait-first composition optimized for phone cards.
- No text, no logos, no watermarks.
