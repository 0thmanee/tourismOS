# AI Media Prompts (Splash + Onboarding)

Use these prompts with Midjourney / SDXL / Flux / Leonardo / Firefly.

## Output Settings (all images)

- **Aspect ratio:** `9:16` portrait
- **Resolution:** `1080x1920` minimum (export 1440x2560 if possible)
- **Color profile:** sRGB
- **Style:** realistic photo / cinematic realism (not illustration/cartoon)
- **Avoid text in image** (no logos, no UI copy, no watermarks)
- **Safe composition:** keep important subjects in center 60% (top & bottom may be covered by mobile UI)

## Brand Style Lock (must be respected)

Use this visual identity in **every** generation:

- **Primary deep forest:** `#0D2A2A` (dark mood / shadow)
- **Secondary forest:** `#1A4040` (mid-tone depth)
- **Warm cream highlights:** `#F5F0E8` (light, premium softness)
- **Terracotta accent:** `#C46A3D` (sunset warmth / cultural accent)
- **Teal accent:** `#1BA7A6` (fresh trusted signal)
- **Muted text-tone influence:** `#4A5C58` (soft neutral balance)

Style language from app/web:

- premium Moroccan travel, authentic, emotionally warm, calm confidence
- cinematic realism, editorial quality, not commercial stock cliché
- elegant contrast: dark forest shadows + terracotta/teal accents + creamy highlights
- no neon palette, no cold blue cast, no oversaturation
- visual simplicity: clear focal subject + breathing space for UI overlays

## Quick generation tips

- If faces/hands are weak, regenerate with: **“natural anatomy, realistic hands, documentary travel photo”**.
- If image feels too busy, add: **“minimal composition, negative space, uncluttered background”**.
- If too dark, add: **“soft balanced exposure, clean highlights, gentle contrast”**.
- Keep all four images color-consistent for a cohesive onboarding experience.
